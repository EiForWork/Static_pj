วิธีการแยกข้อต่อ (Method of joints)

วิธีใช้สำหรับหาแรงในชิ้นส่วนของโครงถักธรรมดาซึ่งประกอบด้วยสภาวะการสมดุลของแรงต่างๆ ที่กระทำต่อสลักยึดข้อต่อที่จุดข้อต่อ (joint) แต่ละจุด ซึ่งก็คือการสมดุลของแรงที่ตัดกันที่จุดๆ หนึ่ง (concurent forees) และใช้สมการของการสมดุลเพียงสองสมการเท่านั้น วิธีนี้จะเริ่มต้นที่ข้อต่อใดก็ได้ที่มีแรงที่ทราบค่าแล้วอย่างน้อยหนึ่งแรง หรือมีแรงที่ไม่ทราบค่าไม่เกินสองแรง การแก้ปัญหาอาจจะเริ่มจากสลักที่ปลาย ด้านซ้ายก่อนและเขียนผังวัตถุอิสระ (F.B.D) ข้อต่อต่างๆ จะกำหนดด้วยตัวอักษร โดยแรงในชิ้นส่วนแต่ละชิ้นจะกำหนดด้วย ตัวอักษรสองตัวที่ปลาย ของชิ้นส่วนนั้นๆ ทิศทางของแรงที่ควรจะเป็นดูได้จากหลักการดังนี้คือ ผัง วัตถุอิสระของเสี้ยวหนึ่งของชิ้นส่วน AF และ AB ได้แสดงไว้ด้วยเพื่อให้ เห็นชัดถึงกลไกของแรงกระทำ (action) และแรงปฏิกิริยา (reaction) จริง ๆ แล้วชิ้นส่วน AB จะสัมผัสทางด้านซ้ายของสลัก ยึด ถึงแม้ว่าจะเขียนแรง AB ไว้ทางด้านขวาและชื้ออกจากสลักยึด ดังนั้นถ้าเขียนหัวลูกศรของแรงไว้ทางด้านเดียวกันกับของสลักยึดที่ชิ้นส่วนแล้วแรงดึง (lension) (เช่นที่ AB) หัวลูกศรจะชี้ออกสลักยึดและแรงกด (compression) (เช่นที่AF) หัวลูกศรจะชี้เข้าหาสลักยึดเสมอ ขนาดของแรง AF หาได้จากสมการ ΣFy =0และ AB หาได้จาก Σ Fx = 0



ต่อไปวิเคราะห์ที่ข้อต่อ F ซึ่งในขณะนี้มีตัวไม่ทราบค่าเพียงสองค่าเท่านั้น คือ EF กับ BF ข้อต่อ B,C,E และ D เป็นลำดับขั้นที่จะวิเคราะห์ต่อไป ผังวัตถุอิสระของแต่ละข้อต่อและแทนรูปหลายเหลี่ยมของแรงในแต่ละข้อต่อ สลักซึ่งเป็นการแสดงสมการของการสมดุล Σ Fx = 0 และ Σ Fy = 0 แบบกราฟฟิก ตัวเลขแสดงถึงลำดับในการวิเคราะห์ต่าง ๆ มีข้อควรสังเกตดังนี้ เมื่อถึงข้อต่อ D ซึ่งอยู่สุดท้ายแล้ว คำนวณหาแรงปฏิกิริยา R2 ซึ่งจะต้องสมดุลกับแรงในชิ้นส่วน CD กับ ED และได้หามาจากข้อต่อที่อยู่ใกล้เคียงมาแล้ว ส่วนนี้เป็นการตรวจสอบความถูกต้องของการคำนวณ และ ในการแยกข้อต่อ C ออกเป็นอิสระจะพบว่า แรงในชื้นส่วน CE จะเป็นศูนย์เมื่อสมการ ΣFy = 0 แต่แรงในชิ้นส่วนนี้จะไม่เป็นศูนย์ถ้ามีแรงภายนอกกระทำที่จุด C อีก บ่อยๆ ครั้งที่จะแสดงแรงดึง T และแรงกด C ของชิ้นส่วนต่าง ๆ ลงในผังโครงถักเริ่มต้น โดยเขียนถูกศรชี้ออกจากข้อต่อเมื่อเป็นแรงดึง และชี้เข้าหาข้อต่อเมื่อเป็นแรงกด



ในบางกรณีไม่สามารถที่จะกำหนดทิศทางที่ถูกต้องของแรงหนึ่งแรงหรือสองแรงที่ไม่ทราบค่าที่กระทำต่อข้อต่อได้เมื่อเป็นเช่นนี้สามารถที่จะกำหนดเช่นไรก็ได้ ค่าลบที่ได้จากการคำนวณแสดงว่าทิศทางที่สมมติไว้ไม่ถูกต้อง ดังนั้นทิศทางที่ถูกต้องจะมีทิศสลับกับที่สมมติไว้ถ้าโครงถักธรรมดามีจุดรองจากแรงภายนอกรับ มากเกินกว่าที่จำเป็นเพื่อให้อยู่ในสภาพสมดุลโครงถักนี้เป็นปัญหาแบบที่ไม่สามารถแก้ปัญหาได้โดยสถิตยศาสตร์แต่เพียงอย่างเดียวได้ (statically indeterminate) และจุดรองรับจากแรงภายนอกมากเกินไป ถ้าโครงถักมีชิ้นส่วนภายในมากเกินกว่าความจำเป็นที่จะป้องกันไม่ให้เกิดการยุบตัว (Collapse) แล้วชิ้นส่วนพิเศษที่เกินมานี้เป็น ชิ้นส่วนเกินภายใน และโครงถักเป็นปัญหาแบบที่ไม่สามารถแก้ปัญหาโดยใช้สถิตยศาสตร์แต่เพียงอย่างเดียวได้ สำหรับโครงถักที่เป็นปัญหาที่สามารถแก้ปัญหาโดยใช้สถิตยศาสตร์ได้ (statically determinate) จะมีความสัมพันธ์ที่แน่นอนระหว่างจำนวนของชิ้นส่วนและจำนวนของข้อต่อที่จำเป็นสำหรับเสถียรภาพ (stabilit) ภายในโดยปราศจากส่วนเกิน (redundancy) เนื่องจากมีสมการของแรงสองสมการในการสมดุลของแต่ละข้อต่อ ดังนั้นจะมี 2j สมการสำหรับโครงถักธรรมดา จะมี j ข้อต่อ ถ้าโครงถักทั้งหมดประกอบด้วย m ชิ้นส่วนรับแรงสองแรง (two-force member) และมีตัวไม่ทราบค่าของแรงปฏิกิริยาที่จุดรองรับมากที่สุดอยู่สามแรงซึ่งจะมีตัวไม่ทราบค่าทั้งหมดเท่ากับ m+3 ดังนั้นโครงถักระนาบ ธรรมดา (simple planc truss) จะประกอบชิ้นส่วนรูปสามเหลี่ยมโดยมีเงื่อนไขตามสมการ m+3 = 2j และพอเพียงสำหรับโครงถักที่สามารถใช้สถิตยศาสตร์แก้ปัญหา ความสัมพันธ์นี้เป็นเงื่อนไขที่จำเป็นสำหรับเสถียรภาพ แต่เงื่อนไขไม่เพียงพอที่จะกำหนดเสถียรภาพได้ เมื่อชิ้นส่วนหนึ่งหรือมากกว่าหนึ่งในจำนวนของ m ชิ้นนี้อาจจะประกอบกันเป็นรูป ที่ไม่เสถียรภาพได้ ถ้า m+3 > 2j แสดงว่ามีจำนวนของชิ้นส่วนมากกว่าจำนวนของสมการอิสระที่มีและโครงถักจะเสถียรภาพแต่เป็นแบบปัญหาที่ไม่สามารถแก้ปัญหาโดยใช้สถิตยศาสตร์ แต่เพียงอย่างเดียวได้ (Statically indeterminate) ถ้า m+3 < 2j โครงถักจะไม่เสถียรภาพเพราะชิ้นส่วนภายในขาดไป และยุบตัวได้เมื่ออยู่ภายใต้แรงกระทำ เงื่อนไขพิเศษสำหรับการวิเคราะห์โครงสร้างถักธรรมดาชนิดพิเศษ เมื่อชิ้นส่วนสองชั้นยุบตัวภายใต้แรงอัดตัวกัน ไและเพิ่มชิ้นส่วนที่สามมาต่อจุดรวมกันเพื่อ ป้องกันการโก่งงอ (bucking) เมื่อรวมแรงในแนวแกน y แล้ว ดังนั้นแรง F3 ชิ้นส่วนที่ 3 จะเท่ากับศูนย์และรวมแรงในแนวแกน x เท่ากับศูนย์จะได้ F1 = F2 โดยแรงมีขนาดเท่ากันแต่มีทิศทางตรงกัน ข้ามมุม θ มีค่าน้อยมาก ถ้าชิ้นส่วนยุบตัวลงด้วยแรงดึงซึ่งจะเกิดขึ้นในแนวทางแกน y แล้ว F3 = 0เมื่อชิ้นส่วนสองชั้นที่ไม่ยุบตัวลง เมื่อชิ้นส่วนไม่มีแรงภายนอกมากระทำที่ข้อต่ออแรงในชิ้นส่วนทั้งสองจะเท่า กับศูนย์ ในขณะชิ้นส่วนยุบตัวสองคู่ต่อกันที่ข้อต่อ แรงในแต่ละคู่มีขนาดเท่ากันแต่มีทิศทางตรงกันข้าม



แผงโครงถักอยู่ในรูปค้ำยันขวาง ซึ่งจะอยู่ในรูปปัญหาซึ่งไม่สามารถแก้ปัญหาได้โดยสถิตศาสตร์แต่เพียงอย่างเดียวได้ ถ้าแต่ละแขนค้ำยันสามารถรับแรงดึงและแรงอัดได้ อย่างไรก็ตามเมื่อแขนค้ำยันอยู่ในรูปยืดหยุ่นได้ ชิ้นส่วนจะไม่สามารถรับแรงอัดได้ เช่น สายเคเบิล จะรับเฉพาะแรงดึงเท่านั้น ซึ่งจะคิดเฉพาะแรงในชิ้นส่วน AB ส่วนใน ชิ้นส่วน CD จะไม่คิด เมื่อเลือกลักษณะแบบนี้จะไม่สามารถตรวจสอบได้อาจจะสมมติว่าแรงดึงที่ได้จากการคำนวณเป็นบวกแสดงว่าถูก แต่ถ้าสมมติว่าแรงดึงกลายเป็นลบ แล้วชิ้นส่วนตรงข้ามจะต้องคำนวณใหม่อีกครั้ง การหาคำตอบของสมการสำหรับแรงสองแรงที่ไม่ทราบค่ากระทำที่ข้อต่อซึ่งอาจจะเลือกแกนอ้างอิง โดยที่ L ทราบค่า ส่วน F1 และ F2 ไม่ทราบค่าผลรวมของแรงตามแนวแกน x ตัดกับแกนอ้างอิง F1 และผลรวมของแรงตามแนวแกน x' ตัดกับแกน F2 เมื่อมุมหาได้ไม่ง่ายนักทำให้การหาคำ ตอบโดยใช้หนึ่งจุดของทิศทางอ้างอิงสำหรับ ถ้าไม่ทราบค่าทั้งสองจะง่ายกว่า
